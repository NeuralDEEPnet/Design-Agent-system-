# Agent Instructions

## Continuous Learning & Memory
As an agent working on Neural DEEP net, you must learn from your mistakes and persist your knowledge.
Whenever you encounter a build error, a logical bug, or a failed approach, you MUST document it.

- Record failures in `docs/learning/failures.md`. Emphasize *why* it failed so we can learn from it.
- Record successful patterns and solutions in `docs/learning/solutions.md`.

Always review these files when encountering similar issues in the future to avoid repeating the same mistakes.
